6. forntend 
    - npm install => install all dependencies
    - npm run build => 
    - sudo apt update => update ubuntu version or dependencies
    - sudo apt install nginx => install nginx which is called as software to deploy frontend
        it make web server, HTTP web server
        give load balancer
        give reverse proxy
        content cache
        mail proxy
    - sudo systemctl start nginx
    - sudo systemctl enable nginx
    - scp -r dist/* /var/www/html => copy files from dist(build files) folder to var/www/html
    - enable port 80 => in AWS security rules add inbound rules
